Description
This bar chart shows the number of user contributions during time.
Here you can study if the contribution pattern is well distributed or if it's composed by spikes.
This statistic is updated every day (max lag time 24h).

How to read it
On the orizontal axis we have time, with one month precision. On the vertical axis the total uploaded files in the category during a month.

How to use it
On the right menu you can order users by name or by number of contributions. If you click into a user name the correspondant graph of contributions will show up.
On the top bar you can also drill-down to a chosen subdirectory.

Use case
Here you can find the community around this category!
